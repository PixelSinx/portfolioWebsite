Visual Basic I

This page was designed to show all of the assignments I did in my first Visual Basic class.

  • Assignment #1
  • Assignment #2
  • Assignment #3

In the first Visual Basic assignment, I created an application that looks through a file, and tells you who the principal is for the school that you selected, along with their phone number.

In the second Visual Basic assignment, I created an application that calculates the total area covered, and cost of tile. This is based on the length, width, and price input into the program.

In the third Visual Basic assignment, I created an application that takes to in to different price varibles for items, and then calculates a total, with a discount, based on the price of each item.

In the fourth Visual Basic assignment, I created an application that calculates your daily calorie limit, based on your gender, activity level, and weight.

In the fifth Visual Basic assignment, I created an application that finds the zip code that you entered in a group, and then tells you what the shipping cost is for that group.
