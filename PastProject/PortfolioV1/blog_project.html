Blog Project

The Blog Project was focused around refining my current knowledge of CSS and teaching me a ton of new tricks along the way. On this project page I will be going over what I learned during each of the homework assignments, and how that effected the website.

Updates

Homework #1

In the first Homework assignment we learned how to build the basic framework of a website. We learned about all of the different headers, lists, and how to style all of the text differently.

Homework #2

In the second homework assignment I added a nav bar and a second webpage that shows how a picture gallery works in CSS, I also added in gradients to the background and sidebar that leads to the blog..

Homework #3

In the third homework assignment I added two more webpages to the website. One that includes a table (On the about page), and another that includes a form (On the contact Page). I also added some nice styles to the nav bar.

Homework #4

In the fourth homework assignment I added responsive web design to the website by allowing the website to resize and move items around to fit any size screen. I also added two more sidebars to the main webpage.

Homework #5

In the fifth homework assignment I added some postioning elements to the nav bar, added print styles to the webpages, moved all internal styles to external style sheets, and updated all comments to the CSS.
