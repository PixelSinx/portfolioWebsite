Portfolio Project

The Portfolio project is a website built around showing off all of the projects that I'm either working, or have worked on. This will allow you to see what skills I have that might benefit you in anyway. There will be separate pages for each individual project, allowing me to go more in depth with updates and news for each project and store all past info easily on the page.

There will also be a personal blog and about me page so you can learn more about me personally, and a page that will allow you to contact me if you desire.
