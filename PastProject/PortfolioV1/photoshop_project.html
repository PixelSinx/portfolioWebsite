Photoshop Project

The Photoshop Project was created to showcase all of the projects I have worked on with Photoshop.

Updates

Project #1

In the first Photoshop Project, I was tasked to create two movie posters. The first poster was supposed to be based around a doomsday scenario, and the second one was supposed to be based around the first U.S. Airborne unit (the 501st). This unit was trained to parachute into battle during World War II.

Project #2

In the second Photoshop Project, I was tasked to create a car montage, and a new cover for the book that we were using for this class.

Project #3

In the third Photoshop Project I was tasked to fix/alter some of my old photos. The first photo had a tree sticking out behind my head, and the second one was pretty dark, so I boosted the lighting.

Project #4

In the fourth Photoshop Project, I was tasked to create a festival postcard, and a hometown postcard.

Project #5

In the fifth Photoshop Project, I was tasked to create a cover for a fall and spring dance recital booklet.

Project #6

In the sixth Photoshop Project, I was taske do display an ad on objects, and also to create an add for a go green campaign.

Project #7

In the seventh Photoshop Project, I was tasked to take a drawing of a house and turn into a colorful model. I was also tasked to design a surfboard.
