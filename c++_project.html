Computer Science(C++)

This page was designed to show all of the assignments I did in my Computer Science(C++) class.

In the first C++ assignment, I wrote a program that calculates the total average income be inputting your average annual income into the program.

In the second C++ assignment, I wrote a program that calculates out the total sales amount, and how much was tax for the month.

In the third C++ assignment, I wrote a program that calculates the total, after the user inputs how many software packages they want, this program also featured error catching, as the user has to enter a number higher than 0.

In the fourth C++ assignment, I wrote a program that takes the MPH and hours traveled, and calculates how far you've traveled for each hours, The program also makes sure that the number you entered is greater than 0.

In the fifth C++ assignment, I wrote a program that takes in 5 test scores, and then displays the average grade.

In the sixth C++ assignment, I wrote a program that checks for a variable in an array using both linear searching, and binary searching.

In the seventh C++ assignment, I wrote a program that takes in test scores into an array that expands as needed, and then calculates the average score.

In the eighth C++ assignment, I wrote a program that takes in a sentence, and then displays it in reverse, uppercase, and finally lowercase.
