Visual Basic II

This page was designed to show all of the assignments I did in my second Visual Basic class.

In the first Visual Basic assignment, I created an application that calculates how many rolls you need for a room, based on the size of the room, and how many square feet the roll covers.

In the second Visual Basic assignment, I created an application that calculates how many students scored in between the minimum and maximum score that you specify.

In the third Visual Basic assignment, I created an application that searches in array to see how many subscriptions the sales person has sold.

In the fourth Visual Basic assignment, I created an application takes sales numbers, and calculates a bonus for each on based on the bonus percentage that is set.

In the fifth Visual Basic assignment, I created an application that allows you to input 3 scores per student, and it then calculates the average score for the student.

In the sixth Visual Basic assignment, I created an application that displays a list of numbers, and everytime the user clicks update, all of the numbers increase by 1.

In the seventh Visual Basic assignment, I created an application that takes in an item number, and then tells you what color the item is.
