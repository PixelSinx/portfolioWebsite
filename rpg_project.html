RPG

This page was designed to show all of the assignments I did in my RPG class.

In the first RPG assignment, I wrote a program that outputs the information stored in a file.

In the second RPG assignment, I wrote a program that checks how much of each product is available, and if it is less than or equal to 20, it outputs that more product is needed.

In the third RPG assignment, I wrote a program that calculates a customers total profits, profit %, and if the customer sales people got a bonus based on the profit amount.

In the fourth RPG assignment, I wrote a program that calculates how many overtime hours an employee has worked, then calculates how much the employee made in overtime pay, and than calculates the final gross pay.

In the fifth RPG assignment, I wrote a program that calculates the amount on hand and compares it to the minimum amount to be kept on hand, it then calculates how many items need to be ordered, and also if any need to be rush ordered. At the end the program totals all of the order amounts up into one big total.

In the sixth RPG assignment, I wrote a program that calculates a customers profits, then calculates 2 different bonuses based on the amount of profit. The program then calculates to figure out if the customer got the maximum amount of bonus on 1, 2, or even none of the bonus amounts.

In the seventh RPG assignment, I wrote a program that calculates a players catches, yards, and yards per catch, while also totaling them all up for each team, the program also calculates if the yards per catch is high enough to warrant the title of all pro receiver.

In the RPG THINK assignment, I wrote a program that checks how much of each item was in stock, on reserve, allocated, on order, and sold. The program then used all of this information, including the minimum allowance, to calculate how much needs to be ordered, and also how much of that needs to be ordered so that it shows up the next day.

In the RPG LAST assignment, I wrote a program that how much a customer made in profit, how much was given out as commission, and also how much of that commission was taxed. The program also didn't show any customers that didn't make over $500 in profit.
