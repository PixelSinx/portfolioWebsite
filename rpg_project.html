Python

This page was designed to show all of the assignments I did in my Python class.

In the first Python assignment, I wrote a program that takes in the cost of a meal, and then outputs the total cost including tax and a tip.

In the second Python assignment, I wrote a program that calculates the yearly cost for your car, after inputing your monthly costs.

In the third Python assignment, I wrote a program that takes 2 primary colors, and outputs the combination color that they make.

In the fourth Python assignment, I wrote a program that takes the MPH and hours traveled, and calculates how far you've traveled for each hours.

In the fifth Python assignment, I wrote a program that takes 5 test scores, shows them alongside their letter grade, and then displays the average grade.

In the sixth Python assignment, I wrote a program that checks to see if an account number is valid, by comparing it to a file.

In the seventh Python assignment, I wrote a program that checks how many lowercase letters, uppercase letters, digits, and spaces are in a file.
